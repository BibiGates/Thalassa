README

Editing

  • CLEAR : reset completely the canvas.
  • UNDO : erase the last drawn segment.
  • REDO : cancel the previous undo.

Symmetry

  • VERTical SYMmetry : apply a vertical symmetry from the center of canvas.
  • HORIZontal SYMmetry : idem with a horizontal symmetry.
  • SET AXIS : allow you to put a visible axis on the canvas by 2 clicks.
  • AXIS SYMmetry : apply the symmetry from your axis.
